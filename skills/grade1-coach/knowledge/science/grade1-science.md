# 科学知识库 — 教科版一年级下册（参考）

> 小学一年级科学下册常见单元，供糯米虾哥哥出题用

## 常见单元（各地教材可能有差异，请根据实际调整）

### 单元1：我们周围的物体
- 轻重比较：哪个重？哪个轻？
- 大小比较：哪个大？哪个小？
- 形状：圆形、方形、三角形
- 物体的特征：颜色、大小、形状、轻重、粗糙/光滑

### 单元2：动物
- 常见的动物：猫、狗、鸡、鸭、鱼、兔子
- 动物的特征：几条腿？有没有毛？生活在哪里？
- 动物和人类的异同
- 动物的家：树上、水里、地上、洞里

### 单元3：植物
- 常见的植物：花、草、树
- 植物的各部分：根、茎、叶、花、果实
- 植物需要什么：阳光、水、空气、土壤
- 种子发芽

### 单元4：天气与季节
- 晴天、阴天、雨天、雪天 ☀️☁️🌧️❄️
- 春夏秋冬的特点 🌸🌻🍂❄️
- 天气对我们的影响

### 单元5：我们的身体
- 身体各部分：头、手、脚、眼睛、耳朵、鼻子、嘴巴
- 五感：看（眼睛）、听（耳朵）、闻（鼻子）、尝（嘴巴）、摸（手）
- 保护身体的方法

### 单元6：水
- 水的特点：无色、无味、透明、会流动
- 水的用途
- 珍惜用水

### 单元7：力与运动
- 推和拉 👐
- 物体运动：快和慢
- 滚、滑、转

## 出题示例

**问答题：**
- "青蛙有几条腿？🐸" → 4条
- "小鸡有没有毛？🐤" → 有
- "太阳从哪边升起？🌅" → 东边
- "一年有几个季节？" → 4个（春夏秋冬）
- "水是什么颜色的？" → 无色

**判断题：**
- "鱼生活在水里 ✅"
- "大象是最小的动物 ❌"
- "雪花是白色的 ✅"
- "植物不需要水 ❌"

**选择题：**
- "下面哪个是鸟类？A.🐱 B.🐦 C.🐟" → B
- "用什么看东西？A.👂 B.👀 C.👃" → B
