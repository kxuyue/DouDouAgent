# 🎒 豆豆的学习系统

## 目录结构
```
workspace-doudou/
├── SOUL.md          ← Agent 性格设定（温暖大哥哥/姐姐）
├── USER.md          ← 豆豆的学习档案（填实际进度）
├── HEARTBEAT.md     ← 每日学习提醒配置
├── skills/
│   └── grade1-coach/
│       ├── SKILL.md ← 核心辅导规则（最重要的文件）
│       └── knowledge/
│           ├── chinese/  ← 语文知识库
│           ├── math/     ← 数学知识库
│           └── english/  ← 英语知识库
├── memory/
│   ├── progress.md     ← 学习进度记录
│   └── review-pool.md  ← 错题复习池（自动维护）
└── README.md
```

## 如何部署

### 方式一：在现有 OpenClaw 中运行
1. 将此 workspace 放到 OpenClaw 的工作区目录
2. 在 openclaw.yaml 中配置一个专门给豆豆用的 Agent
3. 通信渠道用 openclaw-weixin（微信）
4. 启动 Agent

### 方式二：我直接作为豆豆的学习伙伴
如果你暂时不想部署新 Agent，可以让 **我（当前会话）** 兼任豆豆的学习伙伴。
需要你做的就是：
- 把豆豆拉进这个对话 / 让豆豆通过微信和我对话
- 豆豆发语音/文字，我来辅导

## 使用指南

### 豆豆怎么说？
- "学数学" → 开始数学
- "学语文" → 开始语文
- "学英语" → 开始英语
- 直接发答案 → 当前题目继续
- "再来一个" → 加一题 +1⭐
- "不学了" → 结束学习，出小结

### 你怎么看数据？
你可以随时问我：
- "豆豆今天学了什么"
- "豆豆的积分"
- "哪些题做错了"
- "这周学习报告"

## 文件说明

### SOUL.md
定义 Agent 的人格——说话简短、永远积极、每次只做1题、永不扣分、自动切换科目。

### SKILL.md
核心规则：
- 三种学习模式的出题逻辑
- 积分系统的计分规则
- 如何记录学习数据和错题
- 如何根据孩子输入自动判断意图

### 知识库
- **语文**：人教版一年级下册字库、组词、句式
- **数学**：20以内退位减法题库、100以内数、人民币、应用题
- **英语**：朗文 Chapter 3 词汇框架、sight words、问答句型
